
# Список задач
### Описание
**Учебное приложение** список задач с авторизацией.

### Технологии
- Python
- PyQT5
- SQL

### Как запустить
1. Создать и активировать виртаульное окружение
 ```
python -m venv venv
source venv/Scripts/activate
```  
2. Установить зависимости
```
pip install -r requirements.txt
```
3. Запустить скрипт
```
python to_do_list.py
```
